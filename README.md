## 預設規則

| 項目 | 規則 |
|---|---|
| 人數 | 4 人 |
| 發牌 | 52 張，每人 13 張 |
| 起手 | 梅花 3 先出 |
| 點數大小 | 2 > A > K > Q > J > 10 > 9 > 8 > 7 > 6 > 5 > 4 > 3 |
| 花色大小 | 黑桃 > 紅心 > 方塊 > 梅花 |
| 牌型 | 單張、對子、三條、順子、同花、葫蘆、鐵支、同花順 |
| 結束 | 任一玩家出完手牌後結算 |
| 計分 | 贏家加總其他玩家剩餘張數，其他玩家依剩餘張數扣分 |

> 注意：台灣大老二有地方玩法差異。此版先採用常見玩法，之後版本會把規則做成可設定。

## 如何使用

直接打開 `index.html` 即可玩。

若要本機測試規則與對比：

```bash
npm install
npm test
```

本專案目前沒有外部套件，`npm install` 只是建立環境用。

## 如何上傳 GitHub Pages

1. 建立 GitHub repository，例如：`big2-tw`
2. 把此資料夾內的檔案全部上傳到 repository 根目錄
3. 到 GitHub repository：`Settings` → `Pages`
4. Source 選擇 `Deploy from a branch`
5. Branch 選 `main`，資料夾選 `/root`
6. 儲存後等待 GitHub Pages 發布
