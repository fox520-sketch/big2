# v1.0.0 功能凍結政策

## 1.0.x 可接受變更

- 阻礙遊戲進行的錯誤修正
- 資料同步、PWA、手機瀏覽器相容性修正
- Firestore Rules 安全改善
- 效能、無障礙與文字對比改善
- 文件與部署流程修正

## 不直接加入 1.0.x 的內容

- 大型新遊戲模式
- 觀戰與完整回放系統
- 永久帳號、雲端個人資料與跨裝置戰績
- 競技配對、付費功能或廣告
- 需要 Cloud Functions／Blaze 的後端架構

上述功能應進入新的次版本或大版本，並經過獨立測試與遷移規劃。

## 版本規則

- `1.0.x`：穩定維護與錯誤修正
- `1.1.0` 以上：向下相容的小型功能
- `2.0.0`：後端架構、資料格式或規則重大變更
