# v0.7.5 規則與計分設定

## 玩法規則

目前支援 4 種預設：

1. 台灣常用：梅花 3 起手
2. 台灣變體：方塊 3 起手
3. 朋友局：允許 A2345
4. 寬鬆變體：順子可含 2

單人模式切換玩法後會重新開局。多人模式由房主設定，並在「開始多人遊戲 / 下一局」時套用。

## 計分規則

目前支援 4 種預設：

1. 標準：剩幾張扣幾分
2. 8 張以上雙倍
3. 10 張以上雙倍
4. 8 張雙倍、10 張三倍

贏家分數 = 所有輸家扣分總和。輸家分數 = 剩餘張數乘上該規則倍率後扣分。

## 顯示方式

本局結果會顯示：

- 玩家名稱
- 剩餘張數
- 本局分數
- 倍率
- 累計總分
- 勝場
