# v0.6.8 出牌體驗檢查清單

## 手牌排序
- [ ] 依點數排序正常。
- [ ] 依花色排序正常。
- [ ] 依牌型建議排序會把對子、三條、四張同點數集中顯示。
- [ ] 排序不會改變真正手牌內容，只改變畫面顯示順序。

## 出牌提示
- [ ] 選牌後會顯示單張、對子、三條、順子、同花、葫蘆、鐵支、同花順。
- [ ] 非法牌型會顯示白話原因。
- [ ] 上一手存在時，會提示是否能壓過上一手。
- [ ] 第一手會提示必須包含起手牌。

## 推薦出牌
- [ ] 「推薦出牌」會自動選出目前可出的牌。
- [ ] 「最小可出」會選出最小可壓過上一手的牌。
- [ ] 沒有可出牌時會提示建議 Pass。
- [ ] 「清除選牌」可立即取消目前選牌。

## Pass 提醒
- [ ] 有牌可出時按 Pass 會跳出確認。
- [ ] 沒有可出牌時按 Pass 不會多餘提醒。
- [ ] 可關閉 Pass 提醒。

## 手機操作
- [ ] 選中牌會明顯上浮並顯示勾選標記。
- [ ] 操作區在手機上靠近手牌。
- [ ] 手牌可左右滑動。
- [ ] 出牌、Pass、推薦、清除選牌按鈕容易點擊。
