# v0.8.4 正式發布候選版驗收清單

> 自動測試不能取代真實裝置與真人連線。完成後再將版本標記為正式公開版。

## 發布前
- [ ] 保留正式站已填好的 `src/firebase-config.js`
- [ ] 將 v0.8.4 `firestore.rules` 貼到 Firebase Console 並 Publish
- [ ] 確認 GitHub Pages 根目錄可看到 `index.html`、`service-worker.js`、`manifest.webmanifest`
- [ ] 使用無痕視窗確認首頁、隱私頁、離線頁、圖示與分享圖皆可讀取

## PWA
- [ ] Android Chrome 安裝、關閉、重新開啟、更新
- [ ] iPhone Safari 加入主畫面、安全區、鍵盤與重新開啟
- [ ] Windows Chrome／Edge 安裝與更新
- [ ] 「回復上一版」及「恢復最新版」各測試一次
- [ ] 離線可開首頁及單人模式，多人操作會被阻擋

## 多人
- [ ] 2 真人＋2 AI 連續 5 局
- [ ] 3 真人＋1 AI 連續 3 局
- [ ] 4 真人連續 3 局
- [ ] 房主斷線後，其他真人在逾時後取得房主並繼續 AI 回合
- [ ] 同一房主開兩個分頁時，AI 不會重複出牌
- [ ] 下一局保留座位、名稱、總分、勝場且不重複結算

## Firebase
- [ ] 非房內匿名使用者直接修改已存在房間會被拒絕
- [ ] 等待中房間仍可由邀請玩家加入一個座位
- [ ] 離房後 listener 差額回到 0 或 1 以內
- [ ] 過期等待房間可由原房主清理
