# 隱私與資料使用說明

更新日期：2026-06-05

## 本機資料

瀏覽器會使用 localStorage 保存主題、AI 難度、音效、動畫、暱稱、最近房號、教學完成狀態、成就與本機戰績。使用者可從瀏覽器網站設定清除。

## Firebase 多人資料

多人模式使用 Firebase 匿名登入與 Cloud Firestore。房間可能包含匿名 UID、暱稱、房號、座位、牌局狀態、分數、連線時間及房間設定，用於多人同步。

本版：

- 不使用 Cloud Functions。
- 不需要 Blaze。
- 不要求註冊個人帳號。
- 不應在暱稱或房間密碼填入重要個資。

## 第三方服務

- GitHub Pages：網站靜態檔案。
- Firebase：匿名登入與 Firestore 多人同步。
- api.qrserver.com：顯示 QR Code 時，邀請連結會送到該服務產生圖片。

## PWA 快取

Service Worker 會快取遊戲程式、圖示、牌面與必要頁面，以支援快速載入及離線開啟單人模式。多人功能仍需網路。

## 廣告與分析

v0.8.2 沒有內建廣告、行為追蹤或第三方分析程式。

## 公開頁面

網站也提供可直接開啟的 `privacy.html`。
