# v0.8.4 已知限制

1. **朋友休閒版，不是不可繞過的防作弊系統。** 洗牌、發牌與牌局驗證仍在瀏覽器執行；房內成員若刻意改程式或直接操作 Firestore，仍可能作弊。
2. 為了讓玩家輸入房號後加入，所有已匿名登入的使用者可讀取房間文件；目前手牌也在同一房間文件內。正式競技版需要可信任後端與每位玩家獨立手牌文件。
3. 房間密碼是前端雜湊與加入防呆，不應使用其他服務的重要密碼。
4. PWA「回復上一版」只在該裝置仍保有上一版靜態快取時可用；瀏覽器清除網站資料後無法回復。
5. iPhone／iPad 不提供與 Android 完全相同的安裝提示，需由 Safari 分享選單手動加入主畫面。
6. 手機進入深度省電或背景凍結後，Firebase 心跳可能延遲；回到前景後會重建 listener 與心跳。
7. GitHub Pages 是靜態託管，無法在伺服器端自動清理所有人的過期房間。
