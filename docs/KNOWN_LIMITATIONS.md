# v1.0.1 已知限制

## 防作弊

本版不使用 Cloud Functions。洗牌、出牌、Pass、AI 行動與結算由用戶端 JavaScript 搭配 Firestore transaction 執行，因此適合朋友休閒對戰，不適合有獎金或正式競技用途。

## 房間資料可讀性

玩家需先依房號讀取房間才能加入，因此匿名登入者可以讀取房間文件。請勿在暱稱、房間密碼或提示中放入重要個資。

## 房間密碼

房間密碼是朋友局的基本進入保護，不等同伺服器端身分驗證，也不要重複使用其他服務的密碼。

## 房主與 AI

AI 由房主瀏覽器控制。房主裝置長時間休眠、網路品質差或多分頁並行時，可能暫時延遲；系統會以租約、revision 與房主轉移降低重複操作。

## PWA

不同瀏覽器對安裝、背景執行、快取與更新提示的支援不同。iPhone／iPad 需要使用 Safari 的「加入主畫面」，且系統可能在背景清除網站快取。

## Firebase 免費額度

雖然本版不需要 Blaze，Firestore 與 Authentication 仍有用量限制。公開大量使用前應觀察 Firebase Console 的讀寫量。

## 實機驗證

自動測試無法完全模擬 Android、iPhone、手機鎖定、網路切換與 2～4 位真人長時間對戰。正式公開後仍應保留錯誤紀錄與診斷回報流程。
