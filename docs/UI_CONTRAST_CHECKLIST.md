# UI 對比總檢查清單

本文件對應 v0.2.0。目標是避免每個主題在手機、桌機與深色背景下出現「牌看不清楚、按鈕不明顯、文字對比不足」的問題。

## 一、自動檢查標準

執行：

```bash
npm run test:contrast
```

目前自動檢查結果：

```txt
對比檢查通過：13 個主題，169 項色彩組合。最低比值 4.99。
UI 檢查清單通過：9 個區塊。
```
