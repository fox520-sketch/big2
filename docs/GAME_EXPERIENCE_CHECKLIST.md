# v0.6.4 遊戲體驗優化檢查清單

## 排行榜
- 本局結束後，排行榜應顯示玩家名稱、累計總分、勝場與局數。
- 多人房間連續下一局時，排行榜應保留同一房間累計資料。

## 房間列表 / 最近房號
- 建立或加入房間後，該房號會保存到本機最近房號。
- 按「刷新房間列表」可讀取 Firestore 最近房間。
- 若 Firestore 列表讀取失敗，最近房號仍可使用。

## 音效
- 總音效可一鍵開關。
- 可分別關閉選牌、出牌、Pass、勝利、錯誤提示音。
- 音量滑桿會影響所有音效。

## 動畫
- 啟用動畫時，目前回合玩家卡會有淡光提示。
- 關閉動畫時，發牌、出牌、選牌等動畫應盡量停止。
- 系統偏好降低動態效果時，仍會遵守瀏覽器的 reduced motion 設定。

## 手機
- 直向手機會顯示橫向提示。
- 手牌區可左右滑動。
- 結果表格可左右滑動。

## 規則說明頁
- 基本玩法、牌型、Pass / 領出權、計分說明可展開閱讀。
