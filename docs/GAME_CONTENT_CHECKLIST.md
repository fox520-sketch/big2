# v0.7.0 遊戲內容強化清單

## 成就徽章
- 完成第一局後解鎖「初試啼聲」。
- 第一次勝利後解鎖「首勝到手」。
- 今日完成 3 局後解鎖「今日手感熱」。
- 完成多人房間對戰後解鎖「好友開戰」。

## 每日戰績
- 顯示今日局數、勝場、分數、最佳單局與多人局數。
- 資料保存在本機 localStorage。

## 房間密碼
- 建立房間時可選填密碼。
- 加入密碼房時需輸入同一組密碼。
- 這是朋友休閒版保護，不等於正式後端驗證。

## 房主管理
- 房主可踢除玩家。
- 等待中或本局結束時可重新安排座位。
- 牌局進行中踢除玩家時，該座位由 AI 接管到本局結束。

## 遊戲紀錄
- 本機保留最近 40 局。
- 可清除本機遊戲紀錄與每日戰績。

## 音效
- 新增開局音、加入房間音、成就音、座位調整音。
- 可在音效設定中分別開關。
