# v0.6.8 實機測試清單

## 測試目標

v0.6.8 的重點是確認免 Cloud Functions 版在手機與多人實戰時更穩定，尤其是排行榜、最近房號、房間列表、音效動畫開關、出牌 / Pass 防連點與斷線重連。

## 建議測試設備

- 電腦瀏覽器 1 個視窗：房主。
- 手機瀏覽器 1 台：朋友玩家。
- 或同一台電腦使用一般視窗 + 無痕視窗測試兩位真人。

## 必測流程

1. 開啟網頁後確認頁面顯示 v0.6.8。
2. 執行 Firebase 設定檢查，確認 Firebase Config、匿名登入與 Firestore 寫入通過。
3. 房主建立房間。
4. 手機或無痕視窗用邀請連結自動加入。
5. 房主補 AI 空位。
6. 房主開始多人遊戲。
7. Fox 與朋友輪流測試出牌 / Pass。
8. 快速連點出牌 / Pass，確認只會送出一次，並看到防連點提示。
9. 一局結束後確認排行榜更新總分、勝場與局數。
10. 按下一局 / 重新洗牌，確認累計分數保留。
11. 手機鎖螢幕或切換 App，等待一段時間後回來，確認可重新接回座位或由 AI 接管。
12. 刷新房間列表，確認最近房間可以顯示，進行中房間按鈕顯示「回房」。
13. 關閉動畫，確認回合提示仍清楚但手牌不再明顯閃動。
14. 分別關閉選牌、出牌、Pass、發牌、勝利、錯誤提示音，確認各開關有效。
15. 手機直向確認橫向提示出現，手牌可左右滑動。

## 回報問題時請截圖

請盡量截到以下區塊：

- 房間座位列表。
- 房間訊息。
- 多人同步偵錯面板。
- 排行榜。
- 目前回合與上一手。

這樣比較容易判斷是回合同步、Rules、斷線接管、排行榜更新或手機 UI 問題。
