# v0.7.4 規則校正、AI 強化與實戰錯誤檢查清單

## 規則邊界

- [x] 單張依點數與花色比較。
- [x] 對子依對子中最大花色牌比較。
- [x] 三條依點數比較。
- [x] 四張不能直接出，鐵支必須是 4 張同點數加 1 張，共 5 張。
- [x] 台灣常用規則不允許 A2345。
- [x] 朋友局規則允許 A2345，並視為 5 高順。
- [x] 台灣常用規則順子不可含 2。
- [x] 寬鬆變體可允許 J-Q-K-A-2 這類含 2 順子。
- [x] 五張牌型依順子、同花、葫蘆、鐵支、同花順排序。
- [x] 第一手必須包含設定的起手牌，例如梅花 3 或方塊 3。

## 壓牌與 Pass

- [x] 上一手是單張時，只能出更大的單張或 Pass。
- [x] 上一手是對子時，只能出更大的對子或 Pass。
- [x] 上一手是三條時，只能出更大的三條或 Pass。
- [x] 上一手是五張牌型時，只能出合法五張牌型並壓過上一手，或 Pass。
- [x] 新一輪領出者不能 Pass。
- [x] 其他三家都 Pass 後，最後成功出牌者取得領出權。

## AI 強化

- [x] 高難度 AI 評估剩餘手牌時套用目前房間規則，而不是固定使用預設規則。
- [x] 高難度 AI 更少亂拆 2、A、K。
- [x] 高難度 AI 更少亂拆對子、三條與五張牌型。
- [x] 對手只剩 1 張時，高難度 AI 會更積極用較大的合法牌攔截。
- [x] AI 回傳的出牌必須是合法牌型，且必須能壓過上一手。

## 可出牌掃描

- [x] 會計算目前可出幾組牌。
- [x] 會顯示可出牌型摘要，例如單張、對子、順子各幾組。
- [x] 沒有可出牌時，Pass 不需要多餘確認。

## 結算與排行榜

- [x] 本局結果以實際玩家名稱顯示。
- [x] 本局分數依剩餘張數與倍率計算。
- [x] 贏家分數為所有輸家扣分總和。
- [x] 同一 gameNo 不會重複累計總分。
- [x] 排行榜依總分、勝場、座位排序。

## 問題回報

- [x] 多人同步偵錯面板加入「複製偵錯資訊」。
- [x] 偵錯資訊包含版本、房號、狀態、我的座位、目前回合、上一手、gameId、revision、lastActionId。
- [x] 偵錯資訊包含玩家列表、座位列表、最近歷史與瀏覽器資訊。
