# 版本說明

## v0.3.0 - 2026-06-02

### 新增
- 新增 Firebase 多人房間區塊。
- 新增 `src/firebase-config.js`，可貼上 Firebase Web App 設定。
- 新增 `src/firebase-room.js`，處理 Firebase 初始化、匿名登入、建立房間、加入房間、監聽房間、離開房間、AI 補位。
- 新增建立房間功能，產生 6 碼房號。
- 新增加入房間功能。
- 新增邀請連結：`?room=房號&join=1`。
- 新增邀請連結自動加入：Firebase 匿名登入完成後會真正執行 `joinRoom()`。
- 新增 QR Code 顯示，可用手機掃描加入房間。
- 新增房間座位列表，顯示房主、真人、AI、連線狀態。
- 新增房主「補 AI 空位」按鈕。
- 新增 `firestore.rules` 休閒測試版安全規則。
- 新增 `docs/FIREBASE_SETUP.md` Firebase 設定教學。

### 修正
- GitHub Pages 邀請連結會保留目前路徑，不會固定寫死網址。
- 房號輸入會自動轉大寫並移除非英數字元。
- 未填 Firebase 設定時，單人遊戲仍可正常執行，不會因 Firebase 失敗而整頁無法載入。

### 尚未完成
- 多人同步出牌。
- 多人局洗牌與發牌同步。
- Cloud Functions 防作弊驗證。
- 玩家離線後由 AI 接管完整出牌流程。
