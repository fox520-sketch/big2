# 版本說明

## v0.1.1 - 2026-06-02

### 修正

- 修正深色模式中黑桃、梅花牌面使用過亮文字，導致白色牌底上看不清楚的問題。
- 統一所有主題的牌面對比：
  - 黑桃、梅花：深色文字。
  - 紅心、方塊：深紅色文字。
  - 牌底：高亮白底。
  - 牌框：高對比深色或主題色。
- 修正可點擊手牌的 hover / focus 顯示，讓選牌狀態更清楚。

### 新增

- 補齊主題選單：深色模式、電子紙模式、海洋風、護眼風、暮光風、櫻花風、森林風、草原風、夜航風、夕陽風、星空風、糖果風、霓虹風。
- 新增 `scripts/test-contrast.js`，檢查所有主題的主要文字、次要文字、按鈕文字、牌面紅黑花色對比。
- `npm test` 會同時執行規則測試與對比測試。

### 檢查結果

- 規則測試：通過。
- 對比測試：13 個主題、78 項對比檢查通過。

## v0.1.0 - 2026-06-02

### 新增

- 建立台灣大老二規則引擎。
- 支援單張、對子、三條、順子、同花、葫蘆、鐵支、同花順。
- 支援梅花 3 起手。
- 支援點數與花色大小比較。
- 支援單人模式：玩家 1 人 + 3 家簡易 AI。
- 支援出牌、Pass、輪到誰出牌提示。
- 支援一輪結束後由最後出牌者重新領出。
- 支援遊戲結束後分數與排名。
- 支援深色模式、電子紙模式、海洋風。
- 加入手機版排版。
- 加入 README 與 GitHub Pages 上架說明。
- 加入簡易規則測試腳本。

### 已知限制

- AI 目前是簡易版，會選擇最小可出的合法牌。
- 尚未支援 AI 難度 1～20 級。
- 尚未支援 Firebase 多人房間。
- 尚未支援 QR Code 與邀請連結。
- 順子暫不允許包含 2，也不支援 A2345 特殊順。
- 台灣大老二有地方玩法差異，後續版本會加入規則設定。

### 下一版建議

- v0.2.0：加入 AI 難度 1～20 級。
- v0.2.0：完整主題細節美化。
- v0.3.0：加入 Firebase 開房間與加入房間。
