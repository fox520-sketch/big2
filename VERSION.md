# 版本說明

## v0.6.7 - 2026-06-04
### 新增
- 新增「牌局專注模式」：遊戲進行中自動隱藏規則設定、房間列表、Firebase 檢查、偵錯面板、成就、紀錄、規則說明與防作弊摘要等非必要區塊。
- 新增右上角「顯示設定 / 隱藏設定」按鈕，遊戲中可手動展開或收合被隱藏的功能。
- 多人牌局進行中保留必要區塊：狀態列、房間座位、玩家狀態、上一手、出牌操作與我的手牌。
- 單人遊戲進行中也會進入專注模式，減少手機畫面過長與誤觸。

### 修正
- 遊戲中不再顯示大量不相關設定與說明，讓牌桌區更集中。
- 牌局結束或回到等待房間時，自動恢復顯示設定、排行榜、紀錄與說明區塊。
- 手機版專注模式下縮短房間資訊與座位卡間距，提升出牌操作可視範圍。

### 保留
- 仍為免 Cloud Functions 版，不需要 Blaze 付費方案。
- 保留 v0.6.6 的成就徽章、每日戰績、房間密碼、踢除玩家、重新安排座位與遊戲紀錄。

## v0.6.6 - 2026-06-04
### 新增
- 新增成就徽章：首局、首勝、今日 3 局、多人首局、累計 10 局、單局 30 分、累計 5 勝。
- 新增每日戰績：今日局數、勝場、分數、最佳單局與多人局數。
- 強化玩家名稱記憶：暱稱會保存在本機，重新開啟網頁會自動帶入。
- 新增房間密碼：建立房間可設定密碼，加入私人房時需輸入密碼。
- 新增房主管理：房主可踢除玩家；等待中 / 本局結束時可重新安排座位。
- 新增本機遊戲紀錄：保留最近 40 局，記錄時間、房號、排名、分數與勝者。
- 新增更多音效：開局音、加入房間音、成就音、座位調整音，並可分別開關。

### 修正
- 房間列表與最近房號會顯示是否為密碼房。
- Firestore Rules 增加 v0.6.6 的密碼欄位白名單。
- 多人房主操作失敗時會顯示更明確提示。

### 注意
- 房間密碼屬於朋友休閒版保護，密碼雜湊仍儲存在 Firestore 房間文件內，不等同正式後端防作弊。
- 此版仍為免 Cloud Functions 版，不需要 Blaze 付費方案。

## v0.6.5 - 2026-06-03
### 修正
- 強化排行榜顯示：加入最後同步時間、上局排名摘要，避免多人房間切換後看不出資料是否已更新。
- 強化最近房號：最近房號會保留真人 / AI 人數資訊，並自動忽略過舊紀錄，避免列表越來越亂。
- 強化房間列表：進行中房間的按鈕改顯示「回房」，提示同裝置玩家可重新取回座位。
- 修正細部音效設定缺少「發牌音」開關的問題，新增發牌音開關與發牌提示音。
- 優化動畫：手牌不再每次畫面重繪都套用發牌動畫，降低手機上選牌或 Firebase 同步時的閃動感。
- 強化多人出牌 / Pass 防連點：加入本機短時間冷卻，避免手機連按造成重複送出或錯誤提示。
- 強化建立房間穩定性：建立房間時會避免極低機率的房號碰撞覆蓋既有房間。
- 補上真正的規則說明頁區塊，方便在遊戲中查看牌型、Pass、計分與排行榜規則。
- 手機版操作優化：房間按鈕、邀請按鈕與房間列表加入按鈕在小螢幕上改為單欄大按鈕。

### 保留
- 仍為免 Cloud Functions 休閒版，不需要 Blaze 付費方案。
- 保留 v0.6.4 的排行榜、房間列表、最近房號、音效動畫設定、手機橫向提示。
- 保留 v0.6.3 的 Firebase 設定檢查面板與多人同步偵錯面板。

### 測試
- 規則、AI、多人狀態、設定計分、主題對比測試通過。
- JavaScript 語法檢查通過。

## v0.6.2 - 2026-06-02
### 修正
- 強化多人回合同步檢查：出牌 / Pass 會比對 gameId、目前回合與 revision，避免舊畫面或重複點擊送出過期動作。
- 新增出牌 / Pass 防連點：Firebase transaction 完成前鎖定操作按鈕，避免手機連點造成重複送出。
- 強化下一局穩定性：下一局會重新建立 gameId、清空上一局 lastPlay / passCount / winner / results，並保留累計總分。
- 強化斷線重連：已離線玩家若下一局尚未回來，會保留名稱並由 AI 接管；同 UID 回來可取回座位。
- 新增多人同步偵錯面板：顯示版本、房號、我的座位、房主座位、目前回合、revision、lastActionId、gameId 與最後同步時間。
- 保留 v0.6.1 的選牌穩定修正：一般 Firebase 心跳不會清空已選牌。

### 測試
- 規則、AI、多人狀態、設定計分、主題對比測試通過。
- 新增語法檢查與壓縮檔完整性檢查。


## v0.5.0 - 2026-06-02

### 新增
- 新增多人連續對戰流程：一局結束後，房主可按「下一局 / 重新洗牌」。
- 新增同一房間累計總分、勝場與累計總排名。
- 新增玩家離線偵測，真人玩家逾時未回應時會標示離線。
- 新增離線 AI 接管：進行中的牌局若真人玩家離線，該座位暫時改由 AI 出牌。
- 新增房主轉移：房主離線時，會轉移給下一位已連線真人玩家。
- 新增座位卡顯示離線、AI 接管、總分與勝場。
- 新增斷線重連恢復：同一瀏覽器回到房間後，可重新取回原真人座位。

### 修正
- 修正多人結算後無法延續原房間繼續下一局的問題。
- 修正多人累計分數容易被同一局重複計算的問題。
- 修正房主離線後 AI 補位可能停止出牌的問題。

### 注意
- v0.5.0 仍是朋友休閒測試版，正式防作弊版建議下一階段改用 Cloud Functions 洗牌、發牌與驗證出牌。
- 更新此版後，請同步把新版 `firestore.rules` 貼到 Firebase Console 並 Publish。
