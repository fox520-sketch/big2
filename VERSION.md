# 版本說明

## v0.1.0 - 2026-06-02

### 新增

- 建立台灣大老二規則引擎。
- 支援單張、對子、三條、順子、同花、葫蘆、鐵支、同花順。
- 支援梅花 3 起手。
- 支援點數與花色大小比較。
- 支援單人模式：玩家 1 人 + 3 家簡易 AI。
- 支援出牌、Pass、輪到誰出牌提示。
- 支援一輪結束後由最後出牌者重新領出。
- 支援遊戲結束後分數與排名。
- 支援深色模式、電子紙模式、海洋風。
- 加入手機版排版。
- 加入 README 與 GitHub Pages 上架說明。
- 加入簡易規則測試腳本。

### 已知限制

- AI 目前是簡易版，會選擇最小可出的合法牌。
- 尚未支援 AI 難度 1～20 級。
- 尚未支援 Firebase 多人房間。
- 尚未支援 QR Code 與邀請連結。
- 順子暫不允許包含 2，也不支援 A2345 特殊順。
- 台灣大老二有地方玩法差異，後續版本會加入規則設定。

### 下一版建議

- v0.2.0：加入 AI 難度 1～20 級。
- v0.2.0：補齊所有主題風格。
- v0.2.0：建立完整 UI 對比檢查清單。
- v0.3.0：加入 Firebase 開房間與加入房間。
