Big2 TW v0.8.4

隱私與資料使用說明

更新日期：2026-06-06

本機儲存

遊戲會在瀏覽器的 localStorage 儲存主題、AI 難度、音效、動畫、暱稱、最近房號、教學完成狀態、成就與本機戰績。這些資料主要留在你的裝置，可透過清除網站資料移除。

Firebase 多人房間

多人模式使用 Firebase 匿名登入與 Cloud Firestore，可能保存匿名使用者識別碼、暱稱、房號、座位、牌局狀態、分數、連線時間與房間設定。房間資料用於多人同步，不使用 Cloud Functions，亦不要求註冊個人帳號。

第三方服務

  • GitHub Pages：提供網站靜態檔案。
  • Firebase：提供匿名登入與多人房間同步。
  • QR Server：顯示 QR Code 時，邀請連結會送至 api.qrserver.com 產生圖片。

分析與廣告

本版本沒有內建廣告、行為追蹤或第三方分析程式。

本機錯誤紀錄

v0.8.4 會在瀏覽器 localStorage 保留最近 20 筆執行錯誤與診斷上下文，包含錯誤時間、功能、房號、遊戲狀態與網路狀態。這些資料只留在你的裝置，不會自動上傳；可在「錯誤紀錄中心」清除。

PWA 快取

安裝或開啟 PWA 後，Service Worker 會在裝置快取遊戲程式、牌面、圖示與必要頁面，以支援快速載入及離線開啟單人模式。你可以從瀏覽器的網站設定移除快取與 PWA。

房間權限

v0.8.4 Rules 會阻擋非房內成員修改既有房間；但為了透過房號加入，匿名登入者仍可讀取房間資料。本版是朋友休閒版，請勿使用重要個資作為暱稱或密碼。

安全限制

這是朋友休閒版的純前端遊戲。房間密碼與牌局驗證不等同後端安全服務，請勿在暱稱、房號或密碼中使用重要個資或其他服務的密碼。
